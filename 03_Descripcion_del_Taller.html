library(shiny)
library(bslib)

ui <- page_sidebar(
  title = "Mi Primera App Industrial",
  sidebar = sidebar(
    sliderInput("produccion", "Unidades:", 0, 1000, 500)
  ),
  value_box(
    title = "Producción",
    value = textOutput("resultado")
  )
)

server <- function(input, output) {
  output$resultado <- renderText({
    paste(input$produccion, "unidades")
  })
}

shinyApp(ui, server)

11:00 - 11:20 | KPIs Fundamentales en Manufactura

1. OEE (Overall Equipment Effectiveness)

  • Fórmula: OEE = Disponibilidad × Rendimiento × Calidad
  • Componentes: Tiempo disponible, tiempo productivo, unidades buenas
  • Interpretación: World Class OEE ≥ 85%

2. MTBF (Mean Time Between Failures)

  • Fórmula: MTBF = Tiempo Total Operativo / Número de Fallas
  • Aplicación: Mantenimiento predictivo
  • Meta típica: > 200 horas

3. MTTR (Mean Time To Repair)

  • Fórmula: MTTR = Tiempo Total de Reparación / Número de Reparaciones
  • Aplicación: Eficiencia de mantenimiento
  • Meta típica: < 2 horas

4. First Pass Yield (FPY)

  • Fórmula: FPY = (Unidades buenas primera vez / Unidades totales) × 100
  • Meta típica: > 95%

5. Takt Time vs Cycle Time

  • Takt Time = Tiempo disponible / Demanda del cliente
  • Cycle Time = Tiempo real de producción por unidad

Ejercicio 3: Calculadora OEE interactiva (25 min)

# Función para calcular OEE
calcular_oee <- function(disponibilidad, rendimiento, calidad) {
  oee <- disponibilidad * rendimiento * calidad
  
  categoria <- case_when(
    oee >= 0.85 ~ "World Class",
    oee >= 0.65 ~ "Bueno",
    oee >= 0.40 ~ "Regular",
    TRUE ~ "Deficiente"
  )
  
  list(
    oee = oee * 100,
    categoria = categoria,
    disponibilidad = disponibilidad * 100,
    rendimiento = rendimiento * 100,
    calidad = calidad * 100
  )
}
